#Projekt Firegirl and Watergirl

###Kako pokrenuti projekt
Preuzimanjem I raspakiranjem zipanog foldera, ulaz u projekt se ostvaruje otvaranjem projekt.sln te pokretanjem programa u visual studio. Prilikom pokretanja, otvara se prozor sa opcijama Play I Quit, strelicama gore ili dolje se mijenjaju opcije, a pritiskom na tipku enter se pokrene ili zatvori.

###Kontrole u igri
Likovi se kreću različitim tipkama
**Firegirl** -> tipka A-> pokreće se lijevo, tipka W -> skakanje, tipka D -> pokreće se desno, E -> animacija
**Watergirl** -> strelica <- pokreće se lijevo, strelica ↑ skakanje, strelica -> pokreće se desno

###Ovisnosti
**Dijamanti**
Kako bi se razina smatrala završenom, trebaju se prikupiti svi dijamanti,ovisno o boji određeni lik može skupiti dijamant, te se omogućuje izlaz na vrata

**Firepool**-prepreka za Watergirl. Stajanjem na prepreku umire i igra se ponovno pokreće

**Waterpool**-prepreka za Firegirl. Stajanjem na prepreku umire i igra se ponovno pokreće

**Poison spikes**-prepreka za oba lika. Ako bilo tko stane,umiru i igra se ponovno pokreće

###Rješenja za moguće probleme
Pritiskom na tipku R igra se poovno pokreće
